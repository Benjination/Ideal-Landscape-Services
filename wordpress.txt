────────────────────────────────────────
1. HOSTING SELECTION & ACCOUNT SETUP
────────────────────────────────────────
[ ] Choose a managed WordPress host (pick one):
      - SiteGround: ~$100–$180/yr — solid performance, good support
      - Bluehost: ~$100/yr — budget-friendly, beginner-friendly panel
      - Kinsta: ~$300/yr — premium, fastest, best DX but overkill for this site
      Recommendation: SiteGround GrowBig or Bluehost Choice Plus
[ ] Purchase/set up hosting plan
[ ] Point existing domain (ideallandscapeservices.com) DNS to new host
    OR register new domain through host
[ ] Install WordPress via host's one-click installer
[ ] Confirm SSL certificate is active (Let's Encrypt, usually automatic)
[ ] Log into /wp-admin and verify clean install

────────────────────────────────────────
2. SECURITY HARDENING (do this BEFORE building anything)
────────────────────────────────────────
[ ] Change default admin username from "admin" to something non-obvious
[ ] Install and configure security plugin: Wordfence (free tier)
    - Enable login attempt limiting
    - Enable two-factor auth for admin account
    - Turn on firewall
[ ] Change /wp-admin login URL (Wordfence or WPS Hide Login plugin)
[ ] Disable XML-RPC if not needed
[ ] Set file permissions correctly (folders: 755, files: 644)
[ ] Enable automatic WordPress core + plugin updates for security patches

────────────────────────────────────────
3. CORE PLUGIN STACK
────────────────────────────────────────
Install and configure these before touching the theme:

[ ] Elementor (free) or Divi ($89 one-time) — page builder
    Recommendation: Elementor Free + Hello Elementor theme (lightweight, free)
[ ] WPForms Lite — contact forms and newsletter signup
[ ] Smush — image compression and lazy loading
[ ] WP Super Cache or LiteSpeed Cache — page caching for performance
[ ] Yoast SEO — meta tags, sitemaps, search visibility
[ ] UpdraftPlus — automated backups to Google Drive or Dropbox
[ ] Wordfence — (already installed in step 2)
[ ] Smart Slider 3 (free) — hero image slider
[ ] Envira Gallery Lite — gallery page with lightbox
